Solomon's Key

Solomon’s key was the first game I’ve ever worked on and it was written while I was taking the introduction to C# course at Dawson College.

The goal of the project was to replicate the game Solomon’s key in C# using windows forms. I decided to design the character using Samus’ sprites from the game Metroid.

The player can create or destroy cubes to the bottom diagonal that they’re facing. If there are no cubes adjacent to a cube then it will fall down and kill any enemies below it. Cubes can also be pushed to damage enemies as well. Once all the enemies are defeated you've won the game.

The game also featured saving and loading by writing to a file. It was simple to do in a game like this since the board is composed of tiles which are easily stored in an array.

I really enjoyed working on this project because it was my first time animating sprites and getting a character to do what I programmed. It sparked my interest in creating games and I haven’t turned back since.
